Fitness & Wellness Website

Developed a complete Fitness & Wellness web portal that provides users with workout plans, diet recommendations, and basic health guidance. The platform includes a user-friendly interface where users can explore different fitness programs and wellness tips based on their goals. The frontend is built using HTML, CSS, and JavaScript to create a responsive and interactive design, while the backend is developed using Java to handle user authentication and data management. QR Code Generator

Developed a fully functional QR Code Generator web application using Java for backend processing and HTML, CSS, and JavaScript for the frontend interface. This application allows users to enter any text, URL, or custom data, which is then instantly converted into a QR code using a Java-based QR code generation library that encodes the input into a scannable image format. The frontend is designed to be simple, clean, and responsive, where HTML and CSS handle the structure and styling while JavaScript manages user interactions and dynamically displays the generated QR code without page reload. The system ensures fast and accurate QR code generation with a smooth user experience across devices, and this project helped me strengthen my skills in Java backend development, API/library integration, and frontend development using JavaScript.
